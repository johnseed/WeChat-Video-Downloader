# WeChat Video Downloader
微信视频号视频地址获取


### 使用方法

1. 在微信PC端打开视频号
2. 右键打开“查看源代码”
![image](https://user-images.githubusercontent.com/13707525/115578449-e1921080-a2f7-11eb-9a70-23fcf8b78ca9.png)
4. 复制所有源代码输入到[解析页面](./index.html)中，点“解析”，获得视频链接，打开后即可用浏览器下载
